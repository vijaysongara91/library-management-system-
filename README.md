# library-management-system-
📚 Library Management System (Django)

A full-featured Library Management System built using Django. This web application helps manage books, users, and issue/return operations efficiently with a clean interface.

🚀 Features
📖 Add, update, and delete books
👤 User (Student) management
🔄 Issue & return books
🔍 Search and filter books
📊 Track issued and available books
🔐 Admin panel (Django Admin)

🛠️ Tech Stack
💻 Backend: Python, Django
🎨 Frontend: HTML, CSS, js
⚙️ Tools: Django Admin Panel
